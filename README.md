FileMorph

FileMorph is a web-based document conversion platform that enables users to convert, compress, merge, and manage documents quickly and securely.

Features

- Word to PDF
- PDF to Word
- PDF Compression
- Merge PDF
- Split PDF
- Image to PDF
- PDF to Image

Tech Stack

- HTML
- CSS
- JavaScript
- Python (Flask)

Future Enhancements

- OCR Support
- User Authentication
- Cloud Storage
- Premium Features
- Google AdSense Integration
